### Course Overview

This course delves into the exciting field of mixed reality (MR), which blends the virtual and real worlds. Students will learn about the latest MR technologies, devices like the Microsoft HoloLens, and the development of MR applications.

### Expected Curriculum

* **Week 1:** Introduction to Mixed Reality, The MR Spectrum (AR, VR, MR).
* **Week 2:** Mixed Reality Hardware and Software,  Microsoft HoloLens,  Magic Leap.
* **Week 3:** Designing for Mixed Reality,  User Interface and Experience Design.
* **Week 4:** Spatial Computing,  Understanding and Interacting with the Real World.
* **Week 5:** Developing Mixed Reality Applications,  Using MR Development Tools.
* **Week 6:** Exploring Future Trends in Mixed Reality.
