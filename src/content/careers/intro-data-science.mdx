### Course Overview

This introductory course provides a comprehensive overview of the data science pipeline. Students will learn about data collection, cleaning, analysis, and visualization techniques using tools like Python and its associated libraries.

### Expected Curriculum

* **Week 1:** Introduction to Data Science, Data Types, Data Collection Methods.
* **Week 2:** Data Cleaning and Preprocessing, Handling Missing Data, Data Wrangling.
* **Week 3:** Data Analysis Techniques, Descriptive Statistics, Inferential Statistics.
* **Week 4:** Data Visualization, Creating Charts and Graphs, Communicating Insights.
* **Week 5:** Introduction to Machine Learning, Basic Algorithms, Supervised Learning.
* **Week 6:** Data Storytelling, Presenting Findings,  Real-world Applications.
