### Course Overview

This course introduces students to the development of games for augmented and virtual reality platforms. Students will learn how to create immersive and interactive experiences using specialized SDKs and hardware.

### Expected Curriculum

* **Week 1:** Introduction to AR/VR,  AR/VR Hardware and Software.
* **Week 2:**  AR/VR Development Environments, Setting up Your Development Kit.
* **Week 3:**  Creating Interactive AR/VR Experiences, User Interface Design.
* **Week 4:** 3D Modeling and Animation for AR/VR.
* **Week 5:**  AR/VR Game Mechanics, Integrating User Input.
* **Week 6:**  Testing and Deploying AR/VR Games.
