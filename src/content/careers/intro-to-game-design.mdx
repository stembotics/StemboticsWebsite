### Course Overview

This introductory course covers the basic principles of game design, including game mechanics, level design, and storytelling. Students will learn to use either Unity or Unreal Engine to create simple 2D or 3D games.

### Expected Curriculum

* **Week 1:** Introduction to Game Design, Game Mechanics, Game Genres.
* **Week 2:** Introduction to Unity/Unreal Engine, Basic Game Development Concepts.
* **Week 3:** Level Design, Creating Engaging Environments, Game Flow.
* **Week 4:** Game Mechanics Implementation, Scripting Basics.
* **Week 5:** Storytelling in Games, Narrative Design, Character Development.
* **Week 6:**  Game Testing and Polishing, Publishing Your Game.
